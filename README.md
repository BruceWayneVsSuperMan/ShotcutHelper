## 关于你们的提问

### Quicker关联
熟练Quicker的同学应该都知道有按键模拟功能，所以只要模拟Ctrl+space就行了。但大家需要了解到一点，Quicker应用是崇尚鼠标操作的。而ShortCutHelper是崇尚键盘操作的。
